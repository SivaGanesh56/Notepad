# Notepad

### A nextJs fullStack application to create quick notes

## Tech Stack

### NextJs
### MongoDB
### React
### Semantic-UI

## Getting Started

#### clone the repo
git clone https://github.com/SivaGanesh56/Notepad.git

#### install node modules
yarn install

#### First, run the development server
yarn dev
